Condicional e váriaveis

Madeup também temos váriaveis. Váriaveis são objetos que são capazes de armazenar um valor ou uma expressão. Esse valor ou expressão fica armazenado na memória do computador. Para definir uma váriavel, basta escolhermos o nome dela e atribuir um valor. Exemplo: x = 10; nesse exemplo de código, criamos uma váriavel com nome de x que receberá o valor 10.

Além das instruções de movimento e direcionamento, temos outras instruções também. Essas instruções são um utilitário para o programador. Dentre elas, temos a instrução de condicional que é estrturada da seguinte forma:

  • if(se) CONDIÇÃO else(senão) end(fim): o if é uma instrução de comparação. A instrução irá comparar se o parâmetro (CONDIÇÃO) é verdadeiro. Se for verdadeiro, ele irá executar todas as instruções desde a linha de baixo do if, até encontrar o else. Se caso o parâmetro (CONDIÇÃO) for falso, ele irá executar todas as instruções que estão na linha abaixo do else, até encontrar a palavra end, que finaliza a instrução if CONDIÇÃO end.

Nota-se que o valor de x é inicialmente maior que 10, por isso, o programa executurá as instruções do else até o end, se mudarmos o valor de x para 9, por exemplo, ele passará a executar as instruçẽos do if até encontrar o else.
